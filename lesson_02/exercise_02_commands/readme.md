# Docker-compose commands

## Reference

https://docs.docker.com/compose/reference/

## Docker compose commands

Most used docker compose commands

### Detached option

    docker compose <command> -d

### Starting

    docker compose start  

### Stopping
    
    docker compose stop

### Killing

    docker compose kill

### Up (Create + Start)

    docker compose up

### Down (Stop + Remove)

    docker compose down

### Restart

    docker compose restart

### Build

    docker compose build
